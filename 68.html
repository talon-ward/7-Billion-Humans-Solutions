Solutions

Size

   **| jmp01:
   01| if E == "a hole" or
   **|    E == "a wall"
   02|| step W
   03| step SE
   04| if S == "a hole" and
   **|    NW != "a worker" and
   **|    N != "a worker" and
   **|    NE != "a worker"
   05|| tell "everyone" "Goodbye!"
   06| step S
   07| jump jmp01

Speed

   01| forEachDir NW as "mem 2"
   02|| if "mem 2" == "a worker"
   03||| jump jmp02
   **| jmp01:
   04| step SE
   05| "mem 1" = nearest "worker"
   06| if "mem 1" != "a worker"
   07|| tell "everyone" "Goodbye!"
   08| step S
   09| step SW
   10| jump jmp01
   **| jmp02:
   **| jmp03:
   11| step S
   12| step SW
   13| step SE
   14| jump jmp03
The size solution is just taking the commands you need and wrapping them in the proper "if" conditions (or realizing you don't need an "if"). The speed solution uses the fact that only one "if" check needs to be made outside the loop for the purposes of stepping, and only the last worker (or a small group including the last worker) need to check whether to say "Goodbye!" before stepping down.
