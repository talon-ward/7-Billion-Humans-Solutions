Solutions

Size

   01| "mem 1" = nearest "shredder"
   **| jmp01:
   02| step N
   03| if N == "a datacube" and
   **|    N < "mem 2" or
   **|    "mem 2" != "a datacube"
   04|| "mem 2" = set N
   05| if N != "a wall"
   06|| jump jmp01
   07| pickUp "mem 2"
   08| giveTo "mem 1"

Speed

   01| "mem 1" = nearest "shredder"
   02| "mem 2" = nearest "datacube"
   03| step "mem 2"
   04| step N
   05| "mem 3" = nearest "datacube"
   06| if "mem 3" < "mem 2"
   07|| "mem 2" = set "mem 3"
   08| step N
   09| "mem 3" = nearest "datacube"
   10| if "mem 3" < "mem 2"
   11|| "mem 2" = set "mem 3"
   12| step N
   13| "mem 3" = nearest "datacube"
   14| if "mem 3" < "mem 2"
   15|| "mem 2" = set "mem 3"
   16| step N
   17| "mem 3" = nearest "datacube"
   18| if "mem 3" < "mem 2"
   19|| "mem 2" = set "mem 3"
   20| step N
   21| "mem 3" = nearest "datacube"
   22| if "mem 3" < "mem 2"
   23|| "mem 2" = set "mem 3"
   24| step N
   25| "mem 3" = nearest "datacube"
   26| if "mem 3" < "mem 2"
   27|| "mem 2" = set "mem 3"
   28| step N
   29| "mem 3" = nearest "datacube"
   30| if "mem 3" < "mem 2"
   31|| "mem 2" = set "mem 3"
   32| pickUp "mem 2"
   33| giveTo "mem 1"
The size solution is a basic minimum search. The speed solution is unrolling the loop to remove the check for the termination condition.
