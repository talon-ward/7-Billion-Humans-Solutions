Solutions

Size

   **| jmp01:
   01| if S != "a shredder"
   02|| takeFrom S
   03| giveTo E
   04| if "my item" == "a datacube" and
   **|    S == "a shredder"
   05| giveTo S
   06| jump jmp01

Speed

   01| if S == "a printer"
   **|| jmp01:
   02|| if "my item" == "nothing"
   03||| takeFrom S
   04|| jump jmp01
   05| if E == "a worker"
   **|| jmp02:
   06|| if "my item" == "nothing"
   07||| takeFrom W
   08|| jump jmp02
   **| jmp03:
   09| if "my item" == "nothing"
   10|| takeFrom W
   11| else
   12|| giveTo S
   13| jump jmp03
The size solution follows the basic pattern of taking the operations you need and wrapping them in the appropriate conditions. The speed solution is insidiously tricky as it relies on a subtle and secret distinction between the "giveTo" and "takeFrom" commands. Changes that seem like they should have no effect on execution speed can actually make a difference.
