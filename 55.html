Solutions

Size

   01| pickUp S
   **| jmp01:
   02| step N
   03| if H != "nothing" or
   **|    S != "a datacube"
   04|| jump jmp01
   05| forEachDir NW, N, NE, E, SE, S, SW, W as "mem 2"
   06|| "mem 1" = calc "mem 1" + "mem 2"
   07| write "mem 1"
   08| drop

Speed

   01| pickUp S
   02| step N
   03| if NW == "a datacube" and
   **|    NE == "nothing"
   04|| jump jmp01
   05| if N == "nothing" and
   **|    NW == "a datacube"
   06|| jump jmp02
   07| if NW == "nothing" and
   **|    NE == "nothing"
   08|| jump jmp03
   09| jump jmp04
   10| step N
   **| jmp01:
   11| step N
   12| step N
   13| step N
   **| jmp02:
   14| step N
   **| jmp03:
   15| step N
   16| step N
   17| forEachDir NW, N, NE, E, SE, S, SW, W as "mem 2"
   18|| "mem 1" = calc "mem 1" + "mem 2"
   19| write "mem 1"
   20| drop
The size solution just needs to know when you're in the center of a datacube flower. The speed solution isolates the workers, making sure the worker with the longest path has the fewest "if" checks.
